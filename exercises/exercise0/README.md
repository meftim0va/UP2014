Упражнение 1 седмица
============

### Задача 1 ###

Напишете програма, която извежда на екрана някакво текстово съобщението.

#### Примери ####

```c++

-> Hello World!

```

### Задача 2 ###

Напишете програма, която въвежда две числа и извежда техния сбор. 

#### Примери ####

```c++

0 1
-> 1

4 3
-> 7

```
